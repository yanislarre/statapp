# Projet de statistiques appliquées : lien entre endettement et croissance

## Auteurs
Yanis Larré, Alice Israël, Matthieu Humbert, Maxime Gérard (élèves en 2A à l'ENSAE Paris).

## Présentation

Le dépôt présente les codes utilisés pour les statistiques descriptives et la modélisation, accompagnés de descriptions et analyses succinctes. La revue de littérature et autres éléments de sont pas présentés ici. 

## Données utilisées

Les données utilisées dans ce projet sont publiques et proviennent des bases de données de l'OCDE.
